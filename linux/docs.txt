Поздравляю с приобретением одного из лучших комплектов инструментов для OSINT, пентестинга, работы с сетью и т.д.
Тут вы разберётесь с устройством программы и раскроете большую часть её потенциала.

С чего начать?
__________________________

Для начала зайдите в папку с программой.
Для Windows: C:/Eternal
Для Linux: /home/{пользователь}/Eternal

Там вы найдёте файл cfg.json, и выполните следующие шаги:
В параметр database введите путь к папке с базами данных, это требуется чтобы работал поиск по файлам.
В параметр token введите свой токен дискорда, это требуется для работы функций для дискорда. (Я вас уверяю, он не логгируется. Я не настолько отбитый)
В параметры leakosint, usersbox, leakcheck, vkapi, callerapi, введите свои API-ключи к этим сервисам.

Можете также настроить визуал проги:

Темы для параметра color:

- aqua (синий к циановому)
- flame (красный к жёлтому)
- toxic (зелёный к циановому)
- nebula (фиолетовый к синему)
- murder (красный к чёрному)

Темы для параметра result:

- aqua (синий к циановому)
- flame (красный к жёлтому)
- toxic (зелёный к циановому)
- nebula (фиолетовый к синему)
- murder (красный к чёрному)

Темы для параметра colorate:

- vertical
- horizontal
- diagonal
- diagonalbackwards

С настройкой конфига мы закончили.

Теперь зайдём в саму программу. И выполним следующие функции:
1. ГЕНЕРАТОР USERAGENT (Номер: 23)

Эти функции сгенерируют необходимые файлы для работы.
__________________________
